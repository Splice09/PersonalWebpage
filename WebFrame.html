Current Work

September 2015

This is my first current work status update. For the past two months I have been working to put together this resume website. There were two reasons that have headed its creation. First and foremost, is to learn and better understand web development. Second, is to have a practical resume that feels more alive. It's been an enlightening experience so far, and I hope to learn much more as development continues.

Site Specs:

I built the server back-end with Node.js using Connect middleware.

- This was a challenge because there was not much visual feedback for my work, but probably the most educational as far as web development is concerned.

The client side front-end is constructed with HTML5, CSS, and JavaScript

- I tried to be true to form and develop completely from scratch, which is why I avoided boot strap. I wanted to experience the site design first hand, as well as make my own image.

- In light of progress, I currently am using JQuery. I am entertaining the idea of eventually updating this to raw JavaScript. Only to maintain the personal goal of a homemade website.

For scalability and easy updates, I am using Heroku PostgreSQL for the Past Projects page and soon to be Current Work page.

- I wanted some experience with databases, and I felt that this would be the best way to continuously update the page without editing the HTML.
